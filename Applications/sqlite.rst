.. _backbone-label:

sqlite
==============================

Description
~~~~~~~~
SQLite3 is an SQL database engine in a C library. Programs that link the SQLite3 library can have SQL database access without running a separate RDBMS process.

Versions
~~~~~~~~
- Bell: 3.30.1
- Brown: 3.30.1
- Scholar: 3.30.1
- Gilbreth: 3.30.1
- Workbench: 3.30.1

Module
~~~~~~~~
You can load the modules by::

    module load sqlite
