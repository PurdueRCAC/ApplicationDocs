thermocalc
==============================

Description
~~~~~~~~
Thermo-Calc allows you to calculate the state for a given thermodynamic system to obtain insight.

Versions
~~~~~~~~
- Bell: 2019b, 2020a, 2021b
- Brown: 2019b, 2020a, 2021a, 2021b

Module
~~~~~~~~
You can load the modules by::

    module load thermocalc
