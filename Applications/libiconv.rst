Libiconv
==============================

Description
~~~~~~~~
GNU libiconv provides an implementation of the iconv function and the iconv program for character set conversion.

Versions
~~~~~~~~
- Bell: 1.16
- Brown: 1.16
- Gilbreth: 1.16
- Negishi: 1.16
- Anvil: 1.16

Module
~~~~~~~~
You can load the modules by::

    module load libiconv
