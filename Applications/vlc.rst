Vlc
==============================

Description
~~~~~~~~
VLC is a free and open source multimedia player for most multimedia formats.

Versions
~~~~~~~~
- Bell: 3.0.9.2
- Brown: 3.0.9.2
- Gilbreth: 3.0.9.2
- Anvil: 3.0.9.2
- Workbench: 3.0.9.2

Module
~~~~~~~~
You can load the modules by::

    module load vlc
