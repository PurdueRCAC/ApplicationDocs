libv8
==============================

Description
~~~~~~~~
Distributes the V8 JavaScript engine in binary and source forms in order to support fast builds of The Ruby Racer

Versions
~~~~~~~~
- Bell: 3.14
- Brown: 3.14
- Scholar: 3.14
- Anvil: 6.7.17
- Workbench: 3.14

Module
~~~~~~~~
You can load the modules by::

    module load libv8
