Panoply
==============================

Description
~~~~~~~~
Panoply is a Java-based cross-platform NetCDF, HDF and GRIB Data Viewer.

Versions
~~~~~~~~
- Bell: 4.11.6
- Brown: 4.11.0
- Scholar: 4.11.0
- Gilbreth: 4.11.0
- Workbench: 4.11.0

Module
~~~~~~~~
You can load the modules by::

    module load panoply
