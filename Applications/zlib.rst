Zlib
==============================

Description
~~~~~~~~
A free, general-purpose, legally unencumbered lossless data-compression library.

~~~~~~~~

- Brown: 1.2.11-generic, 1.2.11
- Gilbreth: 1.2.11-generic, 1.2.11
- Negishi: 1.2.13

~~~~~~~~

    module load zlib
