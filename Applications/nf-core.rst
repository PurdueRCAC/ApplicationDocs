.. _backbone-label:

nf-core
==============================

Description
~~~~~~~~
A community effort to collect a curated set of analysis pipelines built using Nextflow and tools to run the pipelines.

Versions
~~~~~~~~
- Negishi: 2.7.2, 2.8
- Anvil: 2.7.2, 2.8
- Bell: 2.8

Module
~~~~~~~~
You can load the modules by::

    module load nf-core
