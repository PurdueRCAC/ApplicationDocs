Mpfr
==============================

Description
~~~~~~~~
The MPFR library is a C library for multiple-precision floating-point computations with correct rounding.

Versions
~~~~~~~~
- Bell: 3.1.6
- Brown: 3.1.6
- Gilbreth: 3.1.6
- Negishi: 4.0.2
- Anvil: 4.0.2
- Workbench: 3.1.6

Module
~~~~~~~~
You can load the modules by::

    module load mpfr
