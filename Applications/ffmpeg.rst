Ffmpeg
==============================

Description
~~~~~~~~
FFmpeg is a complete, cross-platform solution to record, convert and stream audio and video.

Versions
~~~~~~~~
- Bell: 4.2.2
- Brown: 4.2.1
- Gilbreth: 4.2.1
- Negishi: 4.4.1
- Anvil: 4.2.2
- Workbench: 4.2.1

Module
~~~~~~~~
You can load the modules by::

    module load ffmpeg
