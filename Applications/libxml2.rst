Libxml2
==============================

Description
~~~~~~~~
Libxml2 is the XML C parser and toolkit developed for the Gnome project but usable outside of the Gnome platform, it is free software available under the MIT License.

Versions
~~~~~~~~
- Bell: 2.9.9
- Brown: 2.9.9
- Scholar: 2.9.9
- Gilbreth: 2.9.9
- Negishi: 2.10.1
- Anvil: 2.9.10

Module
~~~~~~~~
You can load the modules by::

    module load libxml2
