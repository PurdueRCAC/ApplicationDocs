Gmp
==============================

Description
~~~~~~~~
GMP is a free library for arbitrary precision arithmetic, operating on signed integers, rational numbers, and floating-point numbers.

Versions
~~~~~~~~
- Bell: 6.1.2
- Brown: 6.1.2
- Scholar: 6.1.2
- Gilbreth: 6.1.2
- Negishi: 6.2.1
- Anvil: 6.2.1
- Workbench: 6.1.2

Module
~~~~~~~~
You can load the modules by::

    module load gmp
