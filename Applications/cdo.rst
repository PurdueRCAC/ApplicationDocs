Cdo
==============================

Description
~~~~~~~~
CDO is a collection of command line Operators to manipulate and analyse Climate and NWP model Data.

Versions
~~~~~~~~
- Bell: 1.9.5
- Brown: 1.9.5
- Scholar: 1.9.5
- Gilbreth: 1.9.5
- Negishi: 1.9.9
- Anvil: 1.9.9
- Workbench: 1.9.5

Module
~~~~~~~~
You can load the modules by::

    module load cdo
